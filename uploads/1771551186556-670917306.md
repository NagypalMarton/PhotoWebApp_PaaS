# PhotoWebApp_PaaS
Felhőalapú elosztott rendszerek laboratórium (2026) kurzus során egy fényképalbum alkalmazás létrehozása PaaS platformon tetszőleges nyelven.

## Lokális minimál MVP (Docker + MySQL)

Ez a verzió a kért első minimál implementáció:
- felhasználókezelés nélkül,
- lokális MySQL adatbázissal,
- Docker Compose környezetben.

### Funkciók
- Kép feltöltése (név + fájl)
- Képek listázása névvel és feltöltési dátummal
- Rendezés név vagy dátum szerint
- Kép megtekintése listaelemből
- Kép törlése
- 40 karakternél hosszabb név elutasítása
- Opcionális: keresés képnév alapján
- Opcionális: lapozás
- Opcionális: képcímkék megadása és megjelenítése

### Követelmények
- Docker Desktop (Compose támogatással)

### Indítás
```bash
docker compose up --build
```

Az alkalmazás elérhető: http://localhost:3000

### Leállítás
```bash
docker compose down
```

### Adatbázis újrainicializálása (minden adat törlődik)
```bash
docker compose down -v
docker compose up --build
```
